# Consent & Esclations
- Realworld voice agent needs to do more than just answer questions, You need to collect recording consent before capturing calls, which is often legally required. You need to esclate to humans when AI can't help or the user requests
- We can also need to hand off to specifized agent based on the conversational context, so we need to preserve context

### Production workflows:
- Collect recordings consent
- Escalate to humans
- Handoff between agents
- Preserve context
