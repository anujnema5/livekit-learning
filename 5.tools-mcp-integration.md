# Tools and MCP integrations
- Tools let your agent call external APIs, lookup data and take actions on behalf of users.
- WIthout tool agent can only responds on what on your training data and conversation context and that's pretty limiting.
- Tools let you fetch real time information like weather, stock prices or order status.
- They let you take action in external systems like creating tickets, sending emails or updating records.
- You can access your private data from customer databases, updating records getting data from internal docs.
- And you can trigger UI updates on the frontend via RPC tools

## So livekit uses the function tool decorator to expose python functions to the LLM. The LLM decides when to call them based on conversation context.
